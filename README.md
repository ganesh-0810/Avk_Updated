# AVK Wire Traders Website

A modern, responsive business website developed for AVK Wire Traders to showcase fencing products, services, projects, and customer inquiry workflows.

## Features

* Fully responsive design
* Mobile-first layout
* Smooth scrolling navigation
* Interactive service cards
* Project showcase gallery
* Dynamic quote request form
* WhatsApp integration
* Call-to-action sections
* Contact information module
* Optimized image loading
* Custom color palette and branding

## Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript (ES6)

### UI Components

* Font Awesome Icons
* CSS Flexbox
* CSS Grid
* CSS Animations & Transitions

### Hosting

* GitHub Pages
